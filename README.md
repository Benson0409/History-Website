# 🎓 校史資料網站｜History Archive Viewer

這是一個專為呈現這幾年間校史資料所設計的現代化網頁，提供訪客清晰、互動式的歷史瀏覽體驗。專案以 **HTML + CSS + JavaScript** 建構，支援 **RWD 響應式設計**，並以 **橘色系現代風格** 為主軸，讓校園歷史以溫暖且具設計感的方式呈現。

---

## 🔍 特色功能

- **時間軸分類瀏覽**：以主題（如「啟程」、「轉變」、「榮耀」）分類歷史資料，並依年份排序。
- **圖文搭配展示**：每筆資料可包含圖片、公文、事件說明與補充文字。
- **導覽選單**：設計一致的漢堡選單與頁首導覽，支援所有頁面快速切換。
- **響應式設計**：支援手機、平板與桌面裝置。
- **回頂按鈕**：可快速回到頁面頂部。
- **下拉式動畫展開**：點擊年份卡可展開詳細內容，並滑動對齊至中間。
- **模組化結構**：HTML、CSS、JavaScript 檔案分離，方便維護與擴充。
